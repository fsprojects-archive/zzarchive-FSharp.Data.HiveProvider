FSharp.Data.HiveProvider

F# Type Provider for Apache Hive

The Hive Type Provider can be installed from NuGet: 
PM> Install-Package FSharp.Data.HiveProvider

Samples & documentation

The tutorial gives a brief overview of how to use the Hive Type Provider, going through some examples that demonstrate its features. A table of available operators in Hive queries is also accessible.

Contributing and copyright

The project is hosted on GitHub where you can report issues, fork the project and submit pull requests. If you're adding new public API, please also consider adding samples that can be turned into a documentation.
